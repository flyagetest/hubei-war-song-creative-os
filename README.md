# 湖北战歌宇宙 Creative OS

这是为“湖北战歌宇宙”搭建的 Codex 创作操作系统初始仓库。

## 核心目标
把城市战歌、楚超赛事、人物 IP、海报、MV 分镜、歌曲宣发统一到一个可复用、可审查、可持续扩展的创作系统中。

## 默认工作链
1. 创意总监：定义主题、情绪、视觉方向、音乐方向。
2. 连续性审核：核对名称、日期、地点、昵称、颜色、世界观与禁用项。
3. 生产执行：交付歌词、提示词、海报、分镜、短视频脚本、平台文案与资产清单。

## 快速开始
- 新项目：复制 `projects/_template/`
- 新海报：读取 `skills/poster-system/SKILL.md`
- 图像反推：读取 `skills/reverse-prompt/SKILL.md`
- 角色设定板：读取 `skills/character-sheet/SKILL.md`
- MV 分镜：读取 `skills/storyboard/SKILL.md`
- 歌曲宣发：读取 `skills/song-campaign/SKILL.md`

## 文件命名
`YYYYMMDD_城市_项目_用途_比例_v01`

示例：
`20260627_恩施_远征天门_主海报_3x4_v01.png`

## 新增总表
- `projects/README.md`：所有城市、赛事、歌曲项目的状态索引。
- `world/character-bible.md`：阿尼白虎、覃清岚、向长川与未来城市角色规范。
- `world/music-style-bible.md`：曲风矩阵、节奏、口号、歌词结构与禁用表达。
