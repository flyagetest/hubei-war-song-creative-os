# 音乐风格圣经

> 本文件定义“湖北战歌宇宙”的音乐方向、节奏语言、口号逻辑与禁用表达。  
> 具体歌曲仍以对应项目 brief 为准。

## 一、总音乐定位

湖北战歌宇宙不是单一曲风，而是一个以“城市文化 + 足球情绪 + 地方声音 + 群体口号”为核心的音乐体系。

### 四个基础目标
1. **第一耳能喊**：副歌、口号、节奏要适合看台、短视频和现场合唱。
2. **第二耳有地方**：城市地标、方言语感、历史、产业、地貌不能只是名词堆砌。
3. **第三耳有画面**：歌词能直接转化为海报、MV、赛事短片和直播 UI。
4. **第四耳能延展**：每首歌都能拆成战歌版、短视频版、现场群喊版、纪录片版。

## 三、歌词结构规范

### 推荐结构
```text
冷开场 / 号子 / 环境声
主歌 1：城市切口或人物切口
预副歌：压力上升、口号预热
副歌：一句主口号 + 一句画面句
主歌 2：球队、历史、现实困境或远征
桥段 / Rap：信息压缩、节奏变化
最终副歌：全体群喊、可截取短视频
尾声：一句余音、地名或看台回声
```

### 副歌标准
- 主口号最好 4–10 个字。
- 一首歌至少有一个可以在 3 秒内记住的句子。
- 可重复，但每次重复最好有声部、鼓点或文案层级变化。
- 不必强行安排冗长尾副歌；节奏完成时可以直接收束。

## 四、口号系统

### 好口号的标准
- 短：一口气能喊完。
- 硬：有动作感、方向感、力量感。
- 准：必须属于该城市或项目。
- 可视化：能成为旗帜、海报标题、字幕和球迷贴纸。

### 已有口号示例
- 恩施：`客场？照闯！照闯！`
- 恩施：`恩施！恩施！恩施！`
- 宜昌：`宜昌！开闸！`
- 世界杯：`哦大力神杯在发光 / 在旋转`

### 口号写法建议
- 动词优先：开、闯、顶、守、冲、压、起、响、撕、燃。
- 避免所有城市都使用“必胜、无敌、最强”。
- 把城市地貌和动作结合，例如水势、城墙、山路、矿火、江风。

---

## 五、禁用表达与风险控制

### 避免
- 机械沿用上一首歌的句式。
- 连续四字句堆砌，导致歌词像口号清单。
- 只堆地名、景点、历史人物，没有叙事和情绪。
- 过度文雅、没有比赛张力。
- 没有明确人物或场景的空泛热血词。
- 对对手进行无意义侮辱或失控攻击。
- 写入未确认的赛事、比分、球员、年龄、历史事实。

### 对“挑衅感”的处理
- 优先写“我方气势、主场压迫、远征决心、城市力量”。
- 少写纯辱骂，多写“送你回家”“客场照闯”“城门不让”等可传播表达。
- 赛事口号要能用于公开平台和现场喊话。

## 六、声音资产库建议

每个城市项目建议逐步建立以下音频资产：

```text
audio-assets/
├─ crowd-chants/        # 群喊、口号、球迷环境声
├─ city-ambience/       # 江风、雨声、市场、车站、球场灯
├─ folk-textures/       # 地方打击乐、号子、传统乐器纹理
├─ transitions/         # 鼓点、哨声、开闸、电流、旗帜声
└─ stems/               # 主唱、和声、鼓、贝斯、吉他、氛围
```

## 七、平台适配

- **抖音 / 短视频**：前 1–3 秒必须有钩子；优先口号、鼓点、角色或比分画面。
- **小红书**：补充创作缘起、城市细节、海报故事与制作过程。
- **网易云音乐**：强调歌词背景、城市故事、完整制作说明。
- **汽水音乐**：标题明确、情绪强、第一段可迅速进入副歌或核心动机。

## 八、最终自检

交付任何歌曲前，检查：
1. 城市文化是否真实参与了旋律、节奏或歌词，而不是装饰。
2. 副歌是否能被球迷喊出来。
3. 是否至少有一个可剪成 15 秒短视频的爆点。
4. 是否避免沿用上一首的句式和意象。
5. 是否与项目 brief 的口号、禁用项、署名一致。
