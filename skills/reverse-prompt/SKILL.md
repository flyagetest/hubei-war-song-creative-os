# 图像反推 A/B 工作流

## 触发
用户提供参考图，并要求“反推提示词 / 按 A/B 智能体 / 复现画面”。

## 流程
### A：逆向提示词智能体
拆解：
- 主体身份与年龄边界
- 人物面部、发型、服装
- 姿态、视线、动作
- 场景、关键道具
- 构图、景别、镜头、机位
- 光线、色彩、质感、风格
- 负面约束

### B：视觉监督智能体
逐项检查：
- 构图是否对应
- 主体是否明确
- 姿态与镜头是否可复现
- 光线、材质、道具是否遗漏
- 是否有互相冲突的词
- 负面约束是否足够
- 哪些词过于空泛

### 收敛
最多 5 轮；默认只输出最终提示词。
只有用户明确要求时，才展示中间轮次或生成测试图。

## 最终输出格式
1. 正向提示词
2. 负面提示词
3. 参数建议：比例 / n / 镜头感
4. 可选：一行“可替换变量”
